# GOT Next Level + Dragon Dynasty Upgrade

This Vite/React project includes the original cinematic Westeros experience plus a new original **Blood of the Dragon** section inspired by dragon-dynasty fantasy storytelling.

## Added in this version

- New `DragonChronicle` React component
- Original dragon-fight CSS animation
- Interactive succession chapter selector
- Dragon roster / rider dossier selector
- Fire-and-blood inspired dynasty timeline
- Black-vs-green war council map
- Responsive styling for desktop and mobile

## Files added

- `src/components/DragonChronicle.jsx`
- `src/components/DragonChronicle.css`

## Files changed

- `src/App.jsx`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

No copyrighted show footage, Google images, or book text is included. The new experience uses original names, original UI copy, and CSS-based visual effects.
